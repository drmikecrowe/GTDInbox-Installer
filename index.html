GTDInbox-Installer by drmikecrowe

NSIS Installer system for GTDInbox + Prism

This installer leverages the GTDInbox Firefox extension and the Prism XULRunner system to create a standalone GMail client.

Dependencies

http://github.com/gtdinbox/ http://labs.mozilla.com/prism/

Install

To clone the repo: git clone git://github.com/drmikecrowe/GTDInbox-Installer.git Next, pull in the GTDInbox code (there may be a better way, not sure): git submodule init git submodule update Now, edit the install.nsis script with the current version of GTDInbox by changing: !define GTD_VERSION "3.0a19" Compile the install.nsis script.
